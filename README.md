# Long_rotation_periods

Experiments with long-period rotations with TESS.

First is the Hattori long-period periodogram - https://arxiv.org/abs/2505.10376

I have coded this up in the included Jupyter notebook. It reproduces the main plot in their paper and I have spot checked recoverability against ZTF. Match is not perfect, but extremely close. I am confident I have coded this correctly. I still need to implement some slightly better visuals to see what is going on across sectors.
